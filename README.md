### Selenium Test Template for OOTB Hybris Electronics Storefront

This project extended Selenium's Page Object Model creation to make it easier to define Pages, PageTemplates and Components. Although this is primary made to test Hybris OOTB storefront, the core extension of Selenium's creation of Pages will be useful for any project. It heavily simplifies the creation of Page Elements as well as how the pages are structured.
